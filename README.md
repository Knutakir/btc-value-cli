# btc-value-cli
Get the current Bitcoin value in command line

## Installation
```
$ npm install --global btc-value-cli 
```

## Usage
```
$ btc-value --help

        Usage
          $ btc-value

        Options
          --double, -d get value as double

        Examples
        $ btc-value
            $16258
        $ btc-value -d
            $16258.2
```
